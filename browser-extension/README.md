# Rebase Hire Filter

Chrome/Chromium 扩展，用于 `https://hire.rebase.network` 的招聘信息关键字筛选、排序和跳转。

## 安装

1. 使用 Chrome 114 或更新版本，打开 `chrome://extensions`。
2. 开启“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本扩展目录（原项目中为 `browser-extension/`）。
5. 打开 `https://hire.rebase.network` 后点击扩展图标，筛选器会在浏览器侧边栏打开。

## 功能

- 按多个关键字搜索，空格/逗号分隔，所有关键字都需要命中。
- 搜索范围包含标题、公司、地点、薪资、职责、要求、摘要、联系方式等字段。
- 按相关度、可信分、薪资或发布时间排序。
- 点击结果自动加载更多职位并滚动到页面对应条目。
- “只显示匹配”会隐藏当前页面中不匹配的职位。
- 侧边栏在点击主页面后保持打开。
- 搜索词、排序方式和页面筛选状态会持久保存，只有点击“重置”才清空。

## 权限

- `sidePanel`：点击扩展图标时打开持久侧边栏。
- `storage`：保存搜索词、排序和页面筛选状态。
- `activeTab` / `scripting`：与当前招聘页面通信，并在必要时重新注入内容脚本。
- `https://hire.rebase.network/*`：仅在 Rebase 招聘站点读取公开职位数据并操作页面显示。

扩展不包含远程代码，不上传浏览记录或筛选设置。
