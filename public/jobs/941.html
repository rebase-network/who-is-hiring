职位概述

公司/团队介绍 Byterum是一家以AI和区块链为核心技术的公司，立志成为AI与Web3领域的先驱。我们旗下拥有全球知名的NFT数据平台NFTGo，以及正在打造的革命性产品[Minara.ai](https://minara.ai/)——一个支持自然语言交互、实现全流程闭环的AI金融Agent，旨在通过AI技术让所有交易者实现“投资平权”，定义数字金融的未来。

职责描述

  • 1、按照软件规范流程，结合产品研发的实际情况，参与产品测试的全流程，负责产品的质量保证工作；
  • 2、跟踪、协调产品问题的解决，推动问题闭环，持续优化产品质量；
  • 3、负责移动端（iOS/Android）App 的功能测试、兼容性测试、性能测试、安全测试等；
  • 4、编写和维护测试用例、测试报告，确保覆盖率与执行质量；
  • 5、参与测试工具与测试框架的建设与优化，推动测试效率提升。

任职要求

  • 1、本科及以上学历，计算机相关专业；
  • 2、3年以上软件测试经验，其中至少2年以上移动端 App 测试经验（能力突出者条件可放宽）；
  • 3、 熟练掌握黑盒测试方法，能独立设计和执行完整测试用例；
  • 4、熟悉 Git 等版本管理工具，掌握至少一种 Bug 管理工具（Jira、禅道等）和测试用例管理工具（TestLink、XMind 等）；
  • 5、熟悉至少一种编程语言（C/Java/Python等，熟悉 Python 者优先），具备一定的脚本编写与自动化能力；
  • 6、熟悉 iOS 与 Android App 测试流程，掌握多机型、多系统、多网络环境下的兼容性测试；
  • 7、熟练使用常见移动端测试工具（Charles、Fiddler、ADB、Xcode Instruments、Android Studio 等）；
  • 8、具备移动端性能测试经验，能分析内存、耗电、卡顿、网络延迟等问题；
  • 9、 熟悉移动端自动化测试框架（Appium、UIAutomator、XCTest、Airtest、Monkey 等），有自动化实践经验者优先；
  • 10、熟悉 App 上线流程（TestFlight、Google Play、内测分发平台等），了解版本管理规范；
  • 11、细心，责任心强，具备良好的沟通能力和团队协作精神；
  • 12、具备一定的抗压能力和探索精神，能够快速适应新业务；
  • 13、熟悉区块链行业，有相关产品测试经验者优先。
