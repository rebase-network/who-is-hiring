职位概述

Moonveil Entertainment — web3 游戏工作室，获得 Gumi 、HashKey 、Animoca Brands 等知名风投投资。团队成员来自 Riot Games 、Tencent 、Netease 和 Funplus 等公司，办公室位于新加坡，但岗位完全远程。

职责描述

  • 协议开发与基础设施
  • 使用 Solidity （ EVM 网络：Ethereum 、BNB Chain 、Polygon ）和 Rust （ Solana ，可能涉及 NEAR / Polkadot ）开发并优化智能合约
  • 参与游戏机制、代币经济模型和质押功能的链上逻辑设计与实现
  • 处理代理合约（ UUPS 、transparent ）、Gas 优化、不变性测试、模糊测试
  • 研究并实现 ZK 组件（ zk-rollups 、证明、验证）—— 为加分项
  • 区块链数据处理
  • 对 Ethereum 、Solana 、BNB Chain 、Polygon 、HyperEVM 等公链数据进行索引与分析（交易、事件、mempool ）
  • 构建 ETL 管道、数据清洗与存储（ The Graph 、Subgraph 、Dune 、ClickHouse 、TimescaleDB 、PostgreSQL ）
  • 开发自定义索引器和实时 WebSocket 数据流
  • 后端与集成
  • 使用 Node.js / Python / Go / Rust 设计可靠的后端架构
  • 集成钱包（ Wagmi / Viem / Ethers.js / WalletConnect / RainbowKit ）

任职要求

  • 7 年以上商业软件开发经验，其中至少 3 年区块链开发经验 精通 Solidity 和 Rust（如果没有 Solana/Rust 商业经验但 EVM/Solidity 非常扎实，也可以投递，我们主要看重工程能力和学习潜力） 有大规模链上数据索引和处理经验 深入理解 EVM 内部机制（ Gas 、ABI 、存储布局、预编译合约） 英语能力为加分项 了解形式化验证（ Certora 、MIR ）
