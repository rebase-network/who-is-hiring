职位概述

职位描述 我们正在寻找一位在区块链跨链方向有深入研究和实践经验的专家，主导公司跨链相关架构设计与研发工作，推动多链生态的互操作性与应用落地。 作为跨链专家，你将负责： * 设计和实现高性能、安全的跨链通信协议 * 研究不同公链（Ethereum、Polkadot、Cosmos、Solana、Polygon、Layer2 等）的跨链机制，并评估其优劣 * 分析并设计集成主流跨链方案（IBC、轻客户端、跨链桥、消息传递协议等） * 解决跨链安全问题（重放攻击、防欺诈证明、状态一致性） * 指导团队在实际业务中落地跨链方案

职责描述

  • 设计和实现高性能、安全的跨链通信协议

  • 计算机、密码学、分布式系统相关背景，5 年以上开发经验，3 年以上区块链经验。 精通 Polkadot SDK，能够根据需求进行定制化开发。 精通 Rust / Go / Solidity / Move 等至少一种智能合约或底层开发语言，熟悉 Rust 加分。 理解密码学原语（签名、多签、零知识证明、VRF 等），具备安全意识。 有跨链系统实际研发经验者优先（跨链桥、跨链合约、验证器开发等）。 具备良好的架构设计能力与沟通能力，能带领团队攻坚复杂系统。 有联盟链开发经验、熟悉HyperLedger Fabric / FISCO BCOS等框架的开发； 参与过主流跨链项目的开发或贡献。 有 zk 技术、Rollup 等跨链经验。 在跨链相关领域有技术文章、专利或开源项目。 985/211、大厂经验优先 公司简介 Lollipop 正在构建一个互联的区块链世界，让 Solana 开发者无需离开生态就能使用 Polkadot 的多链基础设施，让 Polkadot 开发者能一键访问 Solana 的 DApp 丰富生态，展示 PolkaVM 的无限可能。 X：https://x.com/LollipopHQ 如果你有兴趣，还可以阅读这几篇篇文章： 区块链 “宗师” Gavin Wood 的三次悟道: 解码 JAM 的颠覆性蓝图 https://mp.weixin.qq.com/s/vSSleLMFUpqcduwt4lMTBg 从中国 RISC-V 芯片战略讲起，Open Spec 开放规范的 JAM 正在成为“新技术主权” https://mp.weixin.qq.com/s/PhkeMKDobxCSs6WqGCro5A 连接两大区块链世界：Solana(SVM)-PolkaVM "Lollipop" 计划，获得 Web3 基金会 Grant 资助 https://mp.weixin.qq.com/s/mywu8M7Ily9Jmrjp6_3IYw 请将简历发送至：[melida@lollipop.builders](mailto:melida@lollipop.builders)
