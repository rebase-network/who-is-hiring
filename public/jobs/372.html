职位概述

## 职位名称

职责描述

  • 负责数据分析产品的门户和管理后台的研发工作。
  • 参与产品设计讨论，理解产品需求，确保高质量的代码编写和实施。
  • 与团队协作，持续优化产品性能和用户体验。
  • 负责产品维护和更新，解决技术问题。

任职要求

  • 拥有 3 年以上前端开发经验，能够独立完成前端项目开发。
  • 具备良好的前端架构分析能力与设计能力，能与开发团队保持良好沟通。
  • 有前端工程化思维，有良好的编码习惯。
  • 扎实的 CSS、Html、Javascript 基础。熟悉 TypeScript 优先。
  • 熟悉 Vue 或 React 等现代前端框架，并理解其背后的原理。熟悉 Nuxt.js 或 Next.js 框架。
  • 熟悉 W3C 标准，了解 HTTP 请求的基本原理和流程。
  • 了解各大主流浏览器（Chrome、Edge、Safari）的差异，具有处理兼容问题的经验和能力。
  • 有响应式和适配不同设备的的开发经验。
  • 具备后端开发经验优先。
  • Github 重度用户优先，给开源项目贡献过代码优先。
  • 大专或以上学历。
