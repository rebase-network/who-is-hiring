  • 1、 编程语言:精通 python 或 golang,具备扎实的后端开发基础. 2、 ai 集成经验:具备 llm(大语言模型)应用开发或集成经验,熟悉 openai、claude 等主流 api 调用,或有开源模型(huggingface)的部署调优经验. 3、 架构能力:熟悉微服务架构,熟练使用 redis、kafka/rabbitmq 等中间件处理实时数据流. 4、 行业经验:有量化交易系统开发经验或金融衍生品系统开发经验者优先. 必备条件: 2、精通Python或Golang；熟悉微服务架构，熟练掌握Redis、Kafka/RabbitMQ等中间件；具备LangChain框架使用经验及Prompt *能力。 3、211本科及以上学历2. 3年以上区块链行业经验，必须有DEX（去中心化交易所）、数字钱包或CEX（中心化交易所）的实操产品经验。 优先条件: 1、有区块链、金融衍生品或高并发交易系统设计经验者优先。 2、985/211或海外QS前300，本科以上学历 3、火币/币安/OKcoin/bitgate/gate等平台出身 目标公司/行业: 火币/币安/OKcoin/bitgate/gate等平台出身 至少1年涉及AI大模型（LLM）应用集成或量化交易系统开发经验。
