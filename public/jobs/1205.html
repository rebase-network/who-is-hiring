职位概述

## 公司/团队介绍 快速发展的Crypto中心化交易所，业务稳定，团队偏平化

职责描述

  • 负责公司核心产品的前端架构设计与开发，主导基于 React + Next.js 的技术栈落地；
  • 构建并维护 Monorepo 工程体系，设计高复用性的公共组件库与工具包，提升团队研发效率；
  • 负责实时数据推送场景的技术方案设计与实现，保障 WebSocket 长连接在高并发场景下的稳定性与渲染性能；
  • 持续进行性能工程化优化，包括首屏渲染（ FCP/LCP ）、代码分割、资源懒加载及交互流畅度提升；
  • 与产品、设计、后端团队紧密协作，推动技术方案落地，解决复杂业务场景下的技术挑战。

任职要求

  • 技术深度：扎实的 React 生态功底，深入理解 Hooks 原理及状态管理方案（ Zustand / Redux Toolkit / - Jotai 等），熟练使用 Next.js （ App Router / Pages Router ）进行 SSR/SSG 开发； 工程能力：具备 Monorepo （ Turborepo / Nx / pnpm workspace ）实战经验，能够独立设计并抽离高内聚的公共 Package ，推动前端工程标准化； 实时通信：精通 WebSocket 及长连接管理，熟悉大规模并发数据推送场景下的缓存策略、渲染优化及连接保活机制； 语言基础：深厚的 JavaScript / TypeScript 功底，注重代码的类型安全、可维护性与工程规范； 性能意识：具备前端性能优化方法论，能够针对首屏加载、资源调度、运行时性能进行系统性调优； 软技能：优秀的跨团队沟通与协作能力，具备较强的自我驱动力、Owner 意识及复杂问题拆解能力。 AI 原生开发：具备使用 Claude Code 、Cursor 或 GitHub Copilot 的实战经验，能够编写高质量 Prompt ，善于利用 AI 进行代码库深度检索、理解及大规模重构； 金融背景：有金融产品/交易系统开发经历，熟悉行情数据推送、实时图表渲染或低延迟交互场景；
