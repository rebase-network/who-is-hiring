职位概述

## 公司/团队介绍 * 我们致力于构建去中心化的链上期权交易所; * Syrupal由前 比特大陆, MatrixPort , Top级公链项目 工程师 创立; * 已完成数千万人民币融资。

职责描述

  • 基于Golang相关技术栈的后端开发；
  • 参与链上DEX产品的后端开发，编写**低延迟**后端接口，包括订单簿匹配引擎、交易引擎以及 REST 和 Websocket API；
  • 构建高性能 REST 和 WebSocket API 来为我们的前端页面和程序化交易者提供支持；
  • 协助智能合约工程师进行金融和经济模型生态模型的后端实现；
  • 开发和维护全面的测试套件，包括单元测试、集成测试和端到端测试，以确保后端系统的可靠性。
  • 有DEX/CEX背景优先考虑;
  • 有微服务开发实践，参与分布式高可用、高并发、高性能应用系统开发的优先；
  • 有独立研发项目的能力，从设计、编码到上线、运维，能确保高质量产出的优先。
  • 数字游民

任职要求

  • 计算机相关专业，本科以上学历，3年以上工作经验；
  • 扎实的Go语言基础，良好的编程素养，追求代码美感，有代码洁癖优先；
  • 精通MySQL、NoSQL、Queue等常用存储及中间件，并熟悉相关技术原理；
  • 熟悉Redis，RabbitMQ或Kafka等缓存/中间件的使用；
  • 熟悉分布式系统，微服务架构;
  • 有web3/区块链相关至少1年的开发经验；
  • 独立思考，有产品意识，有blockchain相关的经验优先，能提出系统改善和产品优化者优先。
