职位概述

## 公司/团队介绍 zkLink 是一个多链ZK-Rollup 基础设施，通过零知识证明技术实现不同链上的资产聚合和交易，提供高效、安全、低成本和隐私保护的去中心化交易体验。 zkLink 的愿景是成为连接所有区块链网络的桥梁，让用户可以在任何链上自由地交易任何资产，无需切换网络或钱包，打造一个真正去中心化、无摩擦、无边界的加密世界。 公司创立3年，技术团队实力强劲，目前已融资18.5M USD，资方包括：Coinbase，Alliance DAO，Huobi Ventures，arrington capital等知名基金。 具体JD及其他职位请见：https://zklink.notion.site/Senior-RUST-Engineer-94ab71f4351c428c83cf38b5aa324700

职责描述

  • 使用 RUST 和相关工具开发和维护后端系统。
  • 参与 zkrollup 技术的研究、设计和开发，实现高效的 Layer2 扩容解决方案。
  • 优化系统性能和可靠性，解决技术挑战和问题。
  • 参与代码审查、单元测试和系统集成测试，确保软件质量和稳定性。
  • 持续关注最新的 RUST 和区块链技术发展，不断学习和分享相关知识。
  • 计算机科学或相关领域的学士学位。
  • 至少 3 年的 RUST 开发经验，熟悉 RUST 生态系统和相关工具链。
  • 对区块链技术和 Web3 生态系统有深入的理解和热情。
  • 熟悉 zkrollup 技术或其他 Layer2 扩容解决方案，有相关项目经验者优先。
  • 较强的问题解决和分析能力，能够快速学习和掌握新技术。
  • 出色的团队合作、沟通和协调能力。
  • 居家办公
