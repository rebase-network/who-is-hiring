职责描述

  • 通过合规交易所（如 Coinbase、HashKey、OSL、Sygnum 等）执行 USDT 等数字资产兑换法币操作，确保流程合规、风控可控；
  • 根据公司业务需求，执行境内外法币支付（供应商、市场费用、渠道返佣、薪资等），确保支付准确、及时；
  • 管理公司在 多交易所、多链钱包、托管账户 的资金余额与结构，执行内部资金调拨；
  • 每日/每周/每月资金对账，包括链上账、交易所账、内部账，确保账实一致，如有差异及时跟进；
  • 维护交易记录、审批流与凭证归档，确保资金操作 可追溯 / 可审计；
  • 与 合规 / 风控 / 财务 / 法务 协同，配合提供资金流说明及审计支持；
  • 参与资金流程优化，提高资金周转效率及风险控制标准。

任职要求

  • 本科及以上学历，财务、会计、金融、审计、经济相关专业优先； 必须有加密货币交易所 / 托管机构 / OTC机构相关资金或支付经验（此项为硬性要求）； 熟悉 USDT / USDC / BTC / ETH 转账机制，了解 链上交易、标签地址、提现确认 等操作； 工作严谨、责任心强、执行力高，对数字敏感，具备良好的风险意识；
