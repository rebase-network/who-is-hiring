## Flutter

职责描述

  • 使用 Flutter 开发高性能、可维护的跨平台移动应用（Android 和 iOS）。
  • 参与从需求分析到设计、开发、测试和发布的完整应用程序开发生命周期。
  • 集成第三方 API 和 SDK，优化应用性能和用户体验。
  • 与设计师和产品团队密切合作，确保应用的功能和界面达到最佳水平。
  • 维护和更新现有的应用程序，解决可能出现的问题。
  • 参与代码评审，确保代码质量和团队协作效率。
  • 跟踪 Flutter 和移动开发的最新趋势，并提出创新性解决方案。

任职要求

  • 1. 学历：计算机科学、软件工程或相关专业全日制统招本科（专升本不考虑）及以上学历。 2. 工作经验：3-5 年移动开发经验，至少 2 年 Flutter 实际开发经验。 3. 熟悉 Dart 编程语言以及 Flutter 框架核心功能（如状态管理、路由、动画等）。 4. 有使用 Provider、Riverpod、Bloc 或其他状态管理工具的经验。 5. 熟悉 RESTful API 集成、JSON 数据解析、数据库（如 SQLite、Firebase Firestore）。 6. 熟悉应用性能优化技术，如内存管理、帧率优化、UI 响应速度优化等。 优先考虑： 1. 有数字资产交易所开发经验者优先 2. 熟悉 CI/CD 工具链（如 GitLab CI/CD、Codemagic 等）。 3. 有原生 Android (Kotlin/Java) 或 iOS (Swift) 开发经验优先。 4. 了解敏捷开发方法（Scrum、Kanban）。 5. 具备良好的沟通能力，能够跨团队协作完成任务
