## 公司/团队介绍 ⚙️ 区块链基础设施团队，专注以太坊兼容的高性能区块链系统与底层架构  🤝 由 Dragonfly 领投，投资方与个人支持者包括 Vitalik Buterin  🧠 由深度参与区块链核心系统构建的工程与创始团队带领，工程驱动、长期技术主义  🌍 采用全球远程的工作方式

职责描述

  • 负责 Web3 钱包 、DApp 交互场景的全流程测试：
  • 覆盖助记词/私钥、签名、交易发送、Gas、Nonce、链上状态同步等。
  • 参与需求评审，根据产品逻辑与链上流程 拆解测试点并设计测试方案。
  • 设计 & 执行多端测试：
  • Desktop Wallet
  • Extension Wallet
  • Mobile Wallet
  • 基于 脚本进行 接口测试、链上交互测试。
  • 使用 viem/ethers.js 驱动钱包完成 Swap / 质押 / 借贷 / 抵押 等 DApp 流程的功能验证。
  • 使用工具管理缺陷，推动问题定位、复现、修复及回归验证。
  • 输出测试报告、用户操作文档、验收文档，提出体验优化建议。
  • 能够独立编写测试用例和执行测试。

任职要求

  • 熟悉 MetaMask、Rabby、WalletConnect 等钱包生态。 能阅读基础 Solidity 合约（ERC-20、ERC-721、ERC-4337 等）。 有 Web 前端或测试自动化经验（Playwright/Selenium）。 3年以上Web3 钱包 或 DApp 功能测试经验。 熟悉链上交互流程：签名、Gas、Nonce、链上确认、事件监听、回滚等机制。 了解 DeFi 基础逻辑，如 Swap、借贷、抵押、LP 添加 等。
