## Injective Labs

职责描述

  • 在基于 Cosmos-SDK 的侧链上构建、维护和测试核心组件，包括去中心化订单簿、交易执行协调器、订单匹配器、和 layer-2 执行环境
  • 编写允许用户直接与协议和智能合约交互的开源 Golang 客户端
  • 构建高性能 REST 和 WebSocket API，以为前端交易员和 API 交易员提供支持
  • 参与和执行有关业务未来的战略决策，从对技术基础架构的深入和透彻理解中进行决策
  • 建立、执行和维护与任务和用户需求相一致的技术
  • 为远程工程师团队提供流程、管理、指导和工具，以设计和开发关键任务工程解决方案（例如知识库、教育机会、技术标准和最佳实践）
  • 培养公司在性能、可扩展性和业务基础架构方面的技术敏锐度
  • 确保团队成员能按时交付、并遵守标准流程和安全的最佳实践
  • 与产品团队协作，以设定目标、分配资源并执行
  • 至少拥有在两个大型且高性能的 Golang 代码库工作的经历
  • 对 Go idioms 和 libraries 有深入的了解
  • 对数据结构和 API 设计等 CS 基础知识有深入了解
  • 对区块链的深刻理解
  • 能够与其他技术和非技术团队成员交流技术概念
