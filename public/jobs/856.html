**SynFutures简介** 1. 快速发展的去中心化衍生品协议 (DeFi)，累计交易量超过 3000 亿美元。 2. 由加密货币行业顶尖人才、前华尔街资深人士和区块链技术专家组成的团队。 3. 由包括 Pantera Capital、Polychain、SIG、Framework、Dragonfly、Standard Crypto、 Hashkey Capital 等在内的一流科技投资者支持。 更多信息，请访问我们的网站：[www.synfutures.com](http://www.synfutures.com/) **主要角色介绍** 我们正在寻找一位资深的Web3后端工程师，共同构建下一代去中心化金融（DeFi）基础设施。作为团队的核心成员，您将负责设计并实现高性能、高安全性、低延迟的交易与风控系统，直接处理链上交易、数据索引和资金安全等核心挑战。您的工作将是保障我们产品在激烈市场中的竞争优势和稳定性的基石。 如果您对智能合约交互、套利策略、风险控制和构建可扩展的区块链数据基础设施充满热情，我们非常期待您的加入。 **工作职责** 1. 核心系统开发： 设计、构建和维护可扩展的高性能Web3后端服务，包括但不限于链上交易机器人、自动清算、套利与对冲系统。 2. 风险控制体系： 参与构建核心风控系统，负责设计与开发风险控制模型和实时预警机制，确保资金安全，抵御市场波动和恶意攻击。 3. 数据基础设施： 负责设计与开发Subgraph区块链数据索引，并构建本地中央化数据库，为产品决策、策略回测和实时监控提供强大的数据支持。 4. 架构与产品驱动： 负责新产品及SDK版本的架构设计与开发，与前端开发者和产品经理紧密协作，主导代码开发、测试与部署，积极推动产品上线与迭代。 5. 底层库开发： 参与底层区块链库（如与EVM兼容链交互的SDK）的开发与建设，为上层应用提供稳定、一致且高效的区块链交互接口。 6.第三方集成： 负责与第三方数据平台和服务的集成，设计稳定、高效且安全的API接口服务。 **工作要求** **必备条件：** 1. 学历与经验： 计算机科学或相关工程专业本科及以上学历，具备5年以上后端开发经验，并拥有至少2年直接参与Web3/区块链项目开发的经验。 2. Web3技术栈： 精通Web3技术生态，如 ethers.js, viem, Hardhat, Foundry 等，拥有与智能合约进行深度交互的实战经验。 3. 编程语言： 至少精通以下一门语言：TypeScript (Node.js) 或 Golang。具备使用它们开发高并发、生产级别项目的经验。 4. 云与容器： 熟悉Docker容器技术，拥有基于主流云服务（AWS/GCP/Azure）的开发与部署经验。 5. 系统架构： 熟悉分布式系统与数据库设计，拥有设计和开发大规模高并发系统的经验。精通B/S架构的Web服务和RESTful API设计。 6. 业务与思维： 具备强烈的业务理解力、出色的逻辑思维和解决复杂问题的能力。 7. 团队协作： 优秀的团队协作和沟通能力，拥有强大的执行力和项目迭代意识。主动、自驱，拥有强烈的责任感和主人翁精神。 **优先考虑（加分项）：** 1. 具备 Rust 语言的实战项目经验。 2. 拥有DeFi领域（如DEX, Derivatives等）相关工作或深度参与经验。 3. 有开源项目贡献经验或出色的英语读写能力。 4. 对金融市场、交易策略、风险建模有深入理解。 5. 有开发或维护Subgraph、区块链节点或索引器的经验。
