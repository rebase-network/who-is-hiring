职位概述

公司/团队介绍 公司是一家由行业顶尖力量共同创立，目标是打造下一代AI 驱动的全球投资平台。 企业介绍 创始人来自知名交易所管理层核心位置，长期深度参与全球加密、AI与金融科技生态，建立了覆盖亚洲、美洲与欧洲的顶级国际化资源网络。创始人主导过多项从0到1/从1到10的业务扩张，对产品、增长与策略拥有一线实战经验。 核心团队成员来自腾讯、币安、美团、网易、火币等世界级互联网与加密科技公司，平均拥有超过7年的产品、工程、Al、交易技术领域经验。他们在高并发系统、复杂金融产品、用户增长体系、加密底层协议、推荐算法等方向具备深厚积累。

职责描述

  • 1、 系统集成:负责交易系统的后端架构设计,实现 ai 模型(llm)与交易执行模块的深度集成.
  • 2、 api 开发:设计并维护高可用、低延迟的 restful/websocket api,对接主流金融市场交易接口.
  • 3、 ai 应用落地:负责将ai策略模型工程化,包括 prompt engineering 管理、上下文记忆处理及 agent 决策流的编排(如使用 langchain等框架).
  • 4、 稳定性保障:确保系统在高并发、极端行情下的高可用性与数据一致性.
  • 1.负责移动端产品核心功能模块的 ui 设计及交互原型制作,输出高质量的设计稿、标注及交互动效说明,确保设计方案的可用性与美观度.
  • 2.参与设计系统的搭建与维护,制定统一的颜色、字体、间距、图标及组件规范,提升设计交付效率与多端一致性.
  • 3.针对数据密集、操作高频的场景,通过合理的布局、层级与动效设计,降低用户认知负担,提升信息获取与操作效率.
  • 4.独立完成用户旅程分析与交互流程设计,覆盖正常、异常、边界及极端场景(如网络异常、数据加载失败、操作撤回等),输出可交互原型并推动落地.
  • 5.与产品经理、开发工程师紧密协作,跟进设计实现过程,参与走查验收,确保线上产品与设计稿高度一致.

任职要求

  • 1、 编程语言:精通 python 或 golang,具备扎实的后端开发基础. 2、 ai 集成经验:具备 llm(大语言模型)应用开发或集成经验,熟悉 openai、claude 等主流 api 调用,或有开源模型(huggingface)的部署调优经验. 3、 架构能力:熟悉微服务架构,熟练使用 redis、kafka/rabbitmq 等中间件处理实时数据流. 4、 行业经验:有量化交易系统开发经验或金融衍生品系统开发经验者优先. 高级测试开发工程师 (AI 交易系统方向) Senior Test Development Engineer (SDET) 岗位职责 (Responsibilities) 自动化测试体系搭建： 负责构建交易系统的自动化测试框架（基于 Python/Go），覆盖 API 接口测试、集成测试与端到端（E2E）测试，确保 CI/CD 流程中的质量门禁。 交易核心保障： 针对订单撮合、资产清算、风控策略等核心模块进行极端场景测试（如：高并发下单、行情中断、网络抖动），确保资金数据的绝对一致性。 AI/Agent 专项测试： 设计针对 LLM（大模型）的评估体系，通过构建测试数据集（Dataset）来量化评估 Prompt 优化前后的效果（如准确率、幻觉率）。 测试 AI Agent 在复杂决策流中的逻辑闭环，验证其工具调用（Tool Calling）的正确性。 性能与稳定性测试： 使用 Locust/JMeter 等工具对系统进行压测，分析吞吐量（TPS）与延迟（Latency）瓶颈，协助后端优化性能。 数据质量校验： 验证上游（Twitter/爬虫）数据的清洗质量，监控数据管道的完整性。 任职要求 (Requirements) 技术背景： 计算机相关专业，3 年以上测试开发经验，精通 Python 或 Golang（必须具备读写业务代码的能力）。 自动化经验： 熟练掌握 Pytest、Allure 等测试框架，有 Docker/K8s 环境下的测试部署经验。 能独立搭建CI/CD自动化测试平台。 UI视觉设计师 学历:全日制统招本科及以上 1.211本科及以上学历,设计/美术相关专业,5年以上移动端ui/ux设计经验(需附作品集). 2.有交易类app的设计经验 有从0到1搭建过交易系统设计规范者优先. 3.精通figma(必选)、adobe suite、protopie/axure 擅长使用auto layout和variants搭建动态组件. 4.具备优秀的视觉排版、色彩感知及细节把控能力,能根据不同业务场景输出符合品牌调性的设计方案. ## 工作环境：职场环境，办公楼。
