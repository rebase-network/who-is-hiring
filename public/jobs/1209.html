职位概述

## 公司/团队介绍 Momentum 是基于 SUI 区块链的 ve(3,3) DEX ，由安全的多签托管钱包 MSafe 演变而来。项目由前 Meta Libra 工程师创立，已获得 Coinbase Ventures 、Circle Ventures 、Sui Foundation 等顶级机构超过 1000 万美元的投资

职责描述

  • 1。负责 Momentum 核心区块链基础设施的设计、开发和维护，包括节点部署、链上数据索引、交易监控等；
  • 2。参与 SUI 链上智能合约的集成与部署，协助智能合约工程师完成合约上线和升级；
  • 3。构建和维护链下服务，包括交易路由、流动性聚合、价格预言机、数据缓存等；
  • 4。开发内部工具和脚本，自动化合约部署、测试、监控和报警流程；
  • 5。负责 Sui SDK（TypeScript / Rust）的集成和封装，为前端团队提供可靠的后端服务和 API；
  • 6。搭建和维护 SUI 全节点，确保节点稳定运行，处理链上数据同步和异常恢复；
  • 7。参与系统架构设计，优化整体性能、可扩展性和安全性；
  • 8。编写技术文档，包括架构设计文档、API 文档和运维手册；
  • 9。跟踪 SUI 生态最新技术动态，评估和引入新工具、新框架；
  • 10。与产品、前端和合约团队紧密合作，快速迭代和交付高质量产品。

任职要求

  • 1。5 年以上后端或区块链开发经验，至少 3 年 Web3 / 区块链行业经验； 2。熟悉 SUI 生态，了解 SUI 对象模型、Move 语言基础和 Sui SDK； 3。有 DEX 或 DeFi 协议相关开发经验，理解 AMM、流动性池、滑点计算等核心概念； 4。精通至少一种后端语言（TypeScript / Node.js、Go、Rust、Python 等）； 5。熟悉关系型数据库（PostgreSQL / MySQL）和 NoSQL 数据库（Redis / MongoDB）； 6。有微服务架构和分布式系统开发经验，熟悉 Docker、容器编排和云部署（AWS / GCP / 阿里云）； 7。熟悉 RESTful API 和 WebSocket 开发，有 GraphQL 经验者优先； 8。具备良好的编码习惯，熟悉 Git 工作流和代码审查流程； 9。有链上数据索引经验（如 Subgraph / The Graph / 自定义索引器）优先； 10。了解区块链安全最佳实践，熟悉常见攻击向量； 11。具备独立解决问题的能力，能快速定位和修复线上故障； 12。有良好的团队沟通能力，能与跨职能团队高效协作。 1。有 SUI 链上智能合约开发经验（Move 语言）； 2。有从 0 到 1 的 DeFi 产品上线经验； 3。熟悉其他 L1/L2 生态（Ethereum / Solana / Aptos / Polygon 等）； 4。有高性能交易系统或高频交易系统开发经验； 5。熟悉消息队列（RabbitMQ / Kafka）和事件驱动架构； 6。有 Kubernetes 生产环境运维经验； 7。熟悉监控和日志系统（Prometheus / Grafana / ELK）； 8。有开源项目贡献或技术博客； 9。211 / 985 院校毕业优先； 10。TOEFL 或 IELTS 高分，具备英文沟通能力者优先考虑。 分布式团队，全球化协作 弹性工作时间，结果导向 深度参与 SUI 生态核心协议开发 定期团队线上/线下交流活动
