  • 负责 Skill 自动化生成流水线的端到端测试，包括需求挖掘、Skill 设计到前端生成的全链路验证

  • 建立性能监控与异常告警机制，确保预计算组件和缓存层的稳定性

任职要求

  • 3年以上软件测试经验，有 AI 产品或数据平台测试经验优先 熟悉自动化测试框架（Pytest、Playwright 等），能独立搭建测试工程 具备基础 Python 编程能力，能编写测试脚本和数据验证工具 对金融市场有基本认知（了解 K 线、资金费率、链上数据等概念）优先 有 LLM 应用测试或 Prompt 评估经验者加分 逻辑严谨，能从用户视角发现产品问题，而不只是验证功能是否跑通 了解 Agent Skill 生态（agentskills.io 标准） 有金融数据产品或量化平台的测试经验 熟悉链上数据（Dune、Nansen 等）或宏观金融数据源 直接参与一个定义新品类的早期产品 扁平化团队，你的测试发现能直接影响产品走向 有竞争力的薪资 + 期权激励 远程友好
