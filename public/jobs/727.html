职位概述

职责描述

  • 1、区块链集成与开发：负责 Solana、Tron、TON 等区块链的后端开发，包括链上数据解析、智能合约交互、交易处理等功能。
  • 2、数据解析与存储：设计并优化区块链数据索引和解析方案，确保高效存储和快速查询交易数据、账户信息、NFT 资产等。
  • 3、智能合约交互：开发与智能合约交互的 API 服务，封装智能合约调用，实现链上业务逻辑。
  • 4、高性能后端开发：使用 Go、Rust 或 [Node.js](http://node.js/) 开发高并发、高可用的后端服务，优化链上数据获取和同步方案。
  • 5、分布式架构与数据库：负责链上数据的存储方案，包括使用 PostgreSQL、ClickHouse、Doris、MongoDB 等数据库进行数据存储和分析。
  • 6、Web3 服务与 API 开发：设计并开发 Web3 相关 API，例如钱包服务、DApp 后端、跨链桥对接等。
  • 7、性能优化：优化 RPC 请求、提升节点同步速度、降低存储成本，提高系统稳定性与响应速度。
  • 8、安全与合规：研究和解决私钥管理、权限控制、交易安全等安全性问题，保证系统的合规性和安全性。
  • 9、团队协作：与前端开发、产品经理、智能合约开发团队协作，推动项目落地并优化用户体验。

任职要求

  • 1、编程语言：精通 Go / Rust / [Node.js](http://node.js/) / Python 至少两种，有扎实的计算机基础。
