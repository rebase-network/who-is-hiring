职位概述

职位描述 1、具备知名互联网大厂（如字节/腾讯/阿里/美团等）或知名 Web3 项目背景，能够独立 Hold 住业务、自驱解决问题的架构重构与开发人才。 2、本岗位当前的核心任务是推进系统从 Java 向 Go 的系统级架构改造与代码迁移，并针对多链大数据场景进行深度的性能优化。你将主导高并发、大流量环境下的模块分层与代码重构，确保老业务平稳过渡，新系统高效运行。

  • 1、985/211 硕士学历，3 年及以上 Golang 开发经验，同时具备扎实的 Java 语言基础/阅读能力（能顺畅重构迁移万行级 Java 源码）。 2、深入理解 Go 运行机制（Goroutine、Channel、GC、内存管理），能熟练运用 Go 协程并发与 Channel 优化数据吞吐。 3、 熟练掌握微服务架构、模块分层与系统重构方法论。有扎实的高并发、大流量系统、突发流量削峰及消息队列（MQ）微服务实战经验。 4、 精通 MySQL、PostgreSQL、Redis，有 InfluxDB 等时序数据库使用经验或海量数据批量入库优化经验者优先。 Web3 认知（接受纯传统互联网背景，但需理解原理）： 理解以太坊等公链原理、EVM 机制及 RPC 交互，懂 Solidity 基本逻辑，了解 Geth / Ethers-Go 等交互库，理解 WebSocket 长连接及事件解析。 5、具备严格的单元测试、CI/CD 及基础设施监控意识；熟练使用 AI 编程工具（如 Cursor / Claude Code / OpenCode 等）高效产出。 1、深入参与过 DeFi 协议、NFT Marketplace、跨链桥或 Wallet 项目开发。 2、有大型复杂系统从 Java/其他语言成功重构迁移至 Go 语言的实际落地案例。
