职位概述

## 公司/团队介绍 Momentum 是基于 SUI 区块链的 ve(3,3) DEX ，由安全的多签托管钱包 MSafe 演变而来。项目由前 Meta Libra 工程师创立，已获得 Coinbase Ventures 、Circle Ventures 、Sui Foundation 等顶级机构超过 1000 万美元的投资

职责描述

  • 1。负责 Momentum Dex 核心智能合约的设计与开发，包括 CLMM、ve3,3 模型、流动性激励、投票市场等模块；
  • 2。使用 Move 语言开发 SUI 链上的 DeFi 协议，理解并应用 SUI 对象模型；
  • 3。设计并实现 veTokenomics 机制，包括锁仓、投票权重计算、奖励分配、时间衰减等核心逻辑；
  • 4。编写全面的单元测试和集成测试，使用 Sui Test Framework 确保合约安全性和正确性；
  • 5。进行合约安全审计和代码审查，识别并修复潜在漏洞，遵循安全最佳实践；
  • 6。优化合约性能，减少 Gas 消耗，提升交易执行效率；
  • 7。与前端团队协作，提供合约接口文档，支持 Sui SDK 集成和链上交互；
  • 8。负责合约部署、升级和维护，处理生产环境问题和技术支持。

任职要求

  • 1。3 年以上智能合约开发经验； 2。有至少一项从 0 到 1 的 Web3 产品上线经验，独立负责合约代码交付； 3。深入理解 SUI 生态，熟悉 SUI 对象模型、SDK、工具链和测试框架； 4。有 DeFi 协议开发经验，熟悉 DEX、AMM/CLMM、流动性机制、交易路由等核心功能； 5。熟悉 ve3,3 模型或投票托管代币机制，有相关实现经验优先； 6。熟悉 Solidity 或有以太坊 DeFi 开发经验，理解 DeFi 模式和最佳实践； 7。具备安全意识，了解常见智能合约漏洞（重入攻击、整数溢出、访问控制等），能编写安全的合约代码； 8。能编写全面的测试用例，熟悉单元测试、集成测试和模拟测试； 9。熟悉 Git 版本控制和代码审查流程，有良好的代码规范和文档习惯； 10。具备问题排查和调试能力，能快速定位和解决合约问题； 11。熟悉 Sui SDK（TypeScript）或有前端集成经验，能与前端团队协作。 1。211 或 985 院校毕业优先； 2。有 Move 语言项目相关经验者优先； 3。有 ve3,3 模型或投票托管代币的完整实现经验； 4。参与过智能合约安全审计或漏洞修复； 5。熟悉 Move Prover 形式化验证工具； 6。有 Aptos Move 或其他区块链开发经验（Ethereum、Solana）； 7。熟悉金融数学、算法优化或密码学； 8。有开源项目贡献或技术博客/分享经验； 9。熟悉 SUI 生态的其他协议和工具； 10。TOEFL 或 IELTS 高分，具备英文沟通能力者优先考虑。 分布式团队，全球化协作 弹性工作时间，结果导向 深度参与 SUI 生态核心协议开发 定期团队线上/线下交流活动

  • 月薪：50,000 – 90,000 元/月（视经验而定）
  • 全职，弹性工作制
