项目叙述

**职责描述** - 参与构建和优化基于 Rust 的联盟链开发。 - 开发和维护支持智能合约运行的中间件组件。 - 与团队协作，优化执行性能，确保在资源受限的环境中高效运行。 - 撰写技术文档，记录开发过程中的关键决策和实现细节。 **任职要求 必备条件：** - 精通 Rust 编程语言，具备系统开发能力； - 熟悉区块链技术体系； - 理解虚拟机底层原理，具备 Wasm / RISC-V / sBPF 等开发经验； - 熟悉 Polkadot Substrate / PolkaVM 等架构及其运行机制。 - 具备优秀的远程协作能力和自我驱动能力。 **加分项：** - 有联盟链开发经验、熟悉HyperLedger Fabric / FISCO BCOS等框架的开发； - 对去中心化基础设施、跨链执行架构有深入理解。 - 有Layer2 / Rollup / 零知识证明项目经验者优先。 **项目简介：** Scaling solana Bridgelessly with Network Extension by Polkadot Lollipop 正在构建一个互联的区块链世界，让 Solana 开发者无需离开生态就能使用 Polkadot 的多链基础设施，让 Polkadot 开发者能一键访问 Solana 的 DApp 丰富生态，展示 PolkaVM 的无限可能。
