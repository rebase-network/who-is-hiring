职位概述

远程 - CEX - 风控运营（Risk Operations）

职责描述

  • 负责平台核心风控指标的日常监控与预警，识别刷量套利、代理刷手续费、异常盈亏等行为；
  • 跟踪并分析用户行为数据，发现潜在风险账号，推进用户画像与分级处置机制建设；
  • 配合数据团队搭建风控看板，推动标准化数据体系落地；
  • 监控币种流动性数据、盘口异动、价格剧烈波动，协助制定预警逻辑；
  • 与产品、运营协同，推动奖励系统、限仓机制、黑名单策略的持续优化；
  • 跟进各类风控策略落地效果，持续优化监控逻辑和响应效率；
  • 支持突发事件处理和应急响应，包括异常交易核查与事后复盘；
  • 轮班工作制，配合夜间与节假日风控值班需求（支持远程办公）。

任职要求

  • 本科及以上学历，计算机、金融、数学或相关专业； 3 年以上加密货币交易所或量化团队相关风控、交易运营或数据分析经验； 熟悉主流交易所业务逻辑，了解盘口、撮合、做市、合约杠杆机制等； 熟练使用 SQL / Excel / BI 工具，具备 Python 基础更佳； 具备用户敏感度与风控直觉，能识别潜在风险行为及套利手法； 良好的跨团队沟通能力，能推动项目协作落地； 自我驱动、抗压能力强，能适应快节奏、信息密集的创业环境。 有实际风控脚本配置或风控策略维护经验者优先； 熟悉主流交易所风控系统或监控机制； 有处理高并发、深夜波动场景风控响应经验者优先。 工作模式与薪酬： 提供有竞争力的薪资； 适应敏捷开发环境，可接受创业团队的远程办公与灵活轮班制度（团队时间：北京时间）
