职位概述

## 公司/团队介绍 Top加密货币交易所

职责描述

  • 交易系统架构
  • 负责交易所 KA / COLO 接入架构设计与优化，包括公网、PrivateLink、Peering、专线等接入模式。
  • 负责高频交易场景下的低延迟网络、安全隔离、限流及稳定性治理方案设计。
  • 负责交易链路中的白名单、风控、DDoS、防刷、流量治理等能力建设。
  • 调研并对标行业主流交易所（Binance / OKX / Coinbase 等）在 COLO、大客户、高频交易场景下的架构实现。
  • 推动交易接入层、Gateway、安全组件、网络架构的分层治理与标准化建设。
  • 岗位要求
  • 对交易链路稳定性、tail latency、连接风暴、流量隔离等问题有实际治理经验。
  • 基础架构
  • 1、负责公司业务网关平台（Spring Cloud Gateway/APISIX）的架构设计、开发与持续优化；
  • 2、主导网关层的高可用、高性能、可扩展性方案设计，包括但不限于路由、负载均衡、熔断、限流、安全认证、灰度发布等核心能力的实现与演进；
  • 3、负责公司基础框架的持续迭代；包括但不限于熔断、限流、spring boot升级、中间件各项升级推进；
  • 4、负责中间件系统的性能调优、故障排查和线上问题定位解决；
  • 5、关注开源社区，跟进行业前沿技术并推动公司网关产品能力提升。

任职要求

  • 1、熟练掌握Java/Go/Rust/Lua/Python等一种或多种编程语言； 2、深入理解Spring Cloud Gateway或APISIX等主流网关架构及其工作原理，有生产级别落地经验； 3、深入理解Spring Boot 架构及其工作原理，有二次开发经验； 4、具备分布式系统、微服务架构相关经验，了解服务治理、服务注册与发现等相关概念与实现； 5、对高性能、高可用架构有实际项目经验，有大流量网关优化和稳定性保障经验优先； 6、善于沟通协作，有较强的自我驱动力和技术热情，重视代码质量与团队合作； 7、有开源社区贡献者优先。 正常职场 有交易所、高频交易、量化交易或大型金融交易系统相关经验。 熟悉 ALB/NLB/WAF/API Gateway/Envoy/HAProxy 等接入架构与流量治理方案。 熟悉 websocket、低延迟网络、DDoS 防护、IP 白名单、限流等核心能力设计。 有 AWS / 多云 / IDC 混合架构经验，熟悉 PrivateLink、VPC Peering、专线等网络方案。 对交易链路稳定性、tail latency、连接风暴、流量隔离等问题有实际治理经验。 基础架构 有主流交易所、做市商、量化平台、券商交易系统背景优先。 1、负责公司业务网关平台（Spring Cloud Gateway/APISIX）的架构设计、开发与持续优化； 2、主导网关层的高可用、高性能、可扩展性方案设计，包括但不限于路由、负载均衡、熔断、限流、安全认证、灰度发布等核心能力的实现与演进； 3、负责公司基础框架的持续迭代；包括但不限于熔断、限流、spring boot升级、中间件各项升级推进； 4、负责中间件系统的性能调优、故障排查和线上问题定位解决； 5、关注开源社区，跟进行业前沿技术并推动公司网关产品能力提升。
