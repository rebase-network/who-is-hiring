职位概述

## 公司/团队介绍 Deeper Network 结合网络安全, 网络共享和区块链技术, 旨在构造一个可以容纳数十亿用户的 Web3 基础设施和生态系统。Deeper 的技术, 产品和网络不仅能够极大改善传统互联网用户的上网体验, 同时也为他们参与 Web3 基础设施建设和生态系统铺平了道路。我们拥有多项核心技术，AtomOS 是世界上第一个无锁的网络操作系统。另外 Deeper 独创三叉戟协议、自适应隧道、智能路由、IP 复用以及隧道层拥塞控制等创新技术。DEP 是世界上第一个通用链下去中心化基础执行框架，将为无数新形态应用/领域的诞生提供土壤。目前，Deeper Network 的全球节点已超过7万个，广泛分布于 150+ 国家，而且每个季度都在裂变式增长。

职责描述

  • Substrate Pallet 模块的设计、实现与测试；
  • EVM+DEP 框架开发及应用孵化，探索链下去中心化应用场景；
  • 对探索前沿技术或原理有强烈兴趣；
  • 具有快速学习和沟通表达能力，愿意深入钻研一个领域或问题；

任职要求

  • Rust 区块链研发工程师，有 Rust 开发经验，或与之匹配快速上手 Rust 的能力；
  • Substrate 或其他区块链节点开发经验，熟悉区块链相关原理；
  • 熟悉 Git 版本管理、单元测试与开源协作；
  • 熟悉服务器端开发，了解部署、运维等相关内容；
  • 扎实的英文读写能力；

薪酬说明

  • 25k-50k RMB
  • 月工资；薪资根据工作经验而定，20k - 50k，可以付美元
  • 远程，全时工作
  • 长期，独立合同工（Long Term Independent Contractor）

联系方式

  • [jobs@deeper.network](jobs@deeper.network), 可发中英文简历
  • email:jobs@deeper.network
